# Tea Cafe Authentication

This project is a React Vite application featuring a premium, responsive Login and Sign Up interface with a Tea Cafe theme.

## Features
- **Premium Design**: Warm, tea-inspired color palette with glassmorphism effects.
- **Responsive**: Fully optimized for mobile, tablet, and desktop views.
- **Bootstrap Integration**: Uses React-Bootstrap for layout and components.
- **Lucide Icons**: Beautiful icons for a modern feel.
- **React Router**: Seamless navigation between Login and Sign Up pages.

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Theme Colors
- **Primary**: #6F4E37 (Tea Brown)
- **Secondary**: #C2B280 (Sand)
- **Background**: #F5F5DC (Beige)
- **Accent**: #A52A2A (Auburn)
